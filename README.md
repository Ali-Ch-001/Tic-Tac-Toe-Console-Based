## Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/Ali-Ch-001/Tic-Tac-Toe-Console-Based.git
    cd Tic-Tac-Toe-Console-Based
    ```

2. **Build the project:**

    ```bash
    g++ -o tic_tac_toe main.cpp
    ```
